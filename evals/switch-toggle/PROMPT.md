Build a labeled toggle switch using Base UI's Switch component.

Requirements:
- Import Switch from `@base-ui/react/switch`
- Create a switch with a visible thumb element inside it using `Switch.Root` and `Switch.Thumb`
- Add a label "Enable notifications" using a `<label>` element
- The switch should be off by default
- The switch Root must have `name="notifications"`
- Export the component as default
